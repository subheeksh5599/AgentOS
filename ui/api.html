api

API reference.
REST + SSE runtime.

Runtime

GET
/api/runtime/agents
List all running agents with status, type, and poll interval.
GET
/api/runtime/events
SSE stream of live agent decisions, TXNs, errors, and guardrail hits. text/event-stream.
GET
/api/runtime/history?limit=50
Recent event history as JSON. Optional query param limit (default 50).

State

GET
/api/health
Service health check. Returns service name and version.
GET
/api/state
Full system state: nodes, pools, capacity, transactions, chain verification, agent status.

Console

POST
/api/console
Execute a CLI command on the runtime. Supported: status, pools, nodes, store <mb>, txns, agents, scan, health.

Transactions

GET
/api/transactions
List all transactions in the SHA-256 hash-chained ledger.
POST
/api/transactions
Submit a new transaction. Auto-allocates to least-used storage node. Body: { op: "store", size_gb: 1.0 }

Pools & Nodes

GET
/api/pools
List all storage pools with region, capacity, usage.
POST
/api/pools
Create a new storage pool. Body: { name, region, capacity_tb, tier }
GET
/api/nodes?pool_id=
List storage nodes, optionally filtered by pool_id.
