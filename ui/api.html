Runtime

GET
/api/runtime/agents
Running agents with status, type, poll interval.
GET
/api/runtime/events
SSE stream of agent decisions. text/event-stream.
GET
/api/runtime/history?limit=50
Recent event history as JSON.

State

GET
/api/health
Service health check.
GET
/api/state
Full system state: nodes, pools, capacity, transactions, chain, agents.

/api/pools
