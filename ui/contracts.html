Move contracts.
Seven modules on Sui.

Core

agent_registry.move discovery

Central directory for all deployed agents. Owner binding, category indexing, active/paused toggle. Emits AgentRegistered events for off-chain indexing. Initialized as a shared object on deployment.

public fun register(registry, name, agent_type, strategy_module, wallet_cap, ctx): AgentEntry public fun total_deployed(registry): u64 public fun is_owner(entry, addr): bool
agent_wallet.move guardrails

Per-agent wallet with programmable constraints: max TX value, daily spend limit, whitelisted action types. Only the capability holder can authorize agent actions. Supports pause/resume and guardrail updates.

public fun create(max_tx, daily_limit, allowed_actions, enc_hash, ctx): (Cap, Wallet) public fun agent_transfer(cap, wallet, amount, recipient, action_type, ctx): Coin public fun update_guardrails(cap, wallet, max_tx, daily_limit, actions)
agent_factory.move 1-click

Creates wallet + registers agent + funds + sets guardrails in one Programmable Transaction Block. Emits AgentDeployed event. Designed for the one-click deploy flow from the frontend.

public fun deploy(name, type, strategy, max_tx, daily, actions, funding, enc_hash, ctx)
action_log.move audit

Append-only log with Walrus blob IDs and Sui TXN digests. Each entry is content-addressed. Supports sealing batches to Walrus for permanent storage. Every agent decision is recorded here.

public fun log_action(log, agent_id, action_type, amount, target, ts, walrus_id, txn, ok) public fun seal_batch(log, walrus_blob_id)

Example Strategies

yield_agent.move lp

Yield optimization strategy: pool scanning, APR comparison, rebalancing thresholds (2% minimum edge), compound logic. Configurable min APR and max single-pool exposure (50% default). Kelly-like allocation across pools.

trader_agent.move arbitrage

Arbitrage strategy: order book analysis, stop-loss/take-profit triggers (5%/10%), pair whitelisting, position tracking with signed PnL. 0.5% minimum profit threshold after fees.

prediction_agent.move predict

Prediction market strategy: Kelly criterion sizing, confidence thresholds (60% minimum), edge detection (5% minimum over market-implied). First framework building on DeepBook Predict since May 2026 launch.
