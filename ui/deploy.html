What happens on deploy

01

AgentFactory creates a wallet with your specified guardrails (max TX, daily limit, allowed actions).

02

Wallet is funded with your initial SUI deposit. Factory registers the agent in the on-chain AgentRegistry.

03

Agent entry is emitted as an event. Dashboard picks it up via SSE. Off-chain runtime starts polling.

04

Agent begins autonomous operation — scanning pools, evaluating trades, submitting decisions through Groq LLM.
