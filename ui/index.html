02

How an agent thinks.

01

Fetch state

Sui RPC query for live pool APRs, order book spreads, prediction market odds. Real on-chain data only.

02

LLM reason

Structured context sent to Groq. Llama 3.3 70B returns a JSON decision: which pool, how much, at what confidence.

03

Guardrail check

Max TX, daily limit, allowed actions — validated against on-chain constraints. Blocked actions are logged, never executed.

04

Execute

Valid decisions are packaged into a PTB and submitted. Gas is sponsored — the agent pays nothing.

05

Log to Walrus

Every decision uploaded as a content-addressed blob. Immutable. Verifiable. Permanent.
