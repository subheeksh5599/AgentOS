Seal

Agents poll Sui testnet RPC for live state → send structured context to Groq llama-3.3-70b for reasoning → validate decisions against on-chain guardrails → execute via sponsored PTBs → log immutably to Walrus. Complete lifecycle: strategy to verification.

THE STACK

Content-addressed blobs for agent memory, execution logs. Every decision immutably stored and verifiable on Walrus testnet.

Encrypted agent state and keys. Agents hold funds without exposing signing material on-chain. HSM-grade security.

Unified liquidity across Spot, Margin, Predict markets. First agent framework building natively on DeepBook Predict.

Google OAuth derives Sui address. Agent identity bound to real user. No seed phrase, no wallet extension.

Llama 3.3 70B via Groq API. Structured JSON with confidence scores. Deterministically reproducible decisions.

Gasless sponsored transactions. Multi-step strategies atomically in one PTB. Agent-to-agent payments via AP2.
