§ 02 — How an agent thinks
01

Fetch State

Agent queries Sui testnet for live pool APRs, order book spreads, and prediction market odds. Real on-chain data only. No simulation.

Sui JSON-RPC
02

LLM Reason

Structured context is sent to Groq with the agent's strategy prompt. Llama 3.3 70B returns a JSON decision — which pool, how much, at what confidence.

Groq · json_object mode
03

Guardrail Check

Decision is validated against on-chain constraints: max TX value, daily spend limit, allowed action types. Blocked actions are logged, never executed.

Agent Wallet
04

Execute

Valid decisions are packaged into a Programmable Transaction Block and submitted. Gas is sponsored — the agent pays nothing for transactions.

Sponsored TX · PTB
05

Log to Walrus

Every decision — action, reasoning, hash, timestamp — is uploaded to Walrus as a content-addressed blob. Immutable. Verifiable. Permanent.

§ 03 Sui Primitives Scroll / Drag
01

Walrus

Content-addressed blobs for agent memory, execution logs, and model configuration. Immutable, verifiable audit trail for every agent decision.

02

Seal

Encrypted agent state and private keys at rest. Agents hold funds without exposing signing material on-chain. HSM-grade security.

03

DeepBook

Unified liquidity across Spot, Margin, and Predict markets. Agents trade through a single interface. First framework building on Predict.

04

zkLogin

Google OAuth derives a Sui address. Agent identity is bound to a real user, not a detached keypair. No seed phrase. No wallet extension.

05

Sponsored TXs + PTBs + AP2

Gasless execution via sponsored transactions. Multi-step strategies atomically in one block. Agent-to-agent payments via Google AP2.
