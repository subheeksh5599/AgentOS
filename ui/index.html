Seal

The AgentOS pipeline: agents poll Sui testnet RPC for live state, send structured context to Groq's Llama 3.3 70B for reasoning, validate decisions against on-chain guardrails, execute via sponsored Programmable Transaction Blocks, and log every decision immutably to Walrus. The complete lifecycle — from strategy to execution to verification.

THE STACK

Content-addressed blobs for agent memory, execution logs, and model configuration. Every agent decision is immutably stored and verifiable on Walrus.

Encrypted agent state and private keys at rest. Agents hold funds without exposing signing material on-chain. HSM-grade security for autonomous execution.

Unified liquidity across Spot, Margin, and Predict markets. Agents trade through a single interface. The first agent framework building natively on DeepBook Predict.

Google OAuth derives a Sui address. Agent identity is bound to a real user, not a detached keypair. No seed phrase. No wallet extension needed.

Llama 3.3 70B reasoning via Groq's high-performance API. Structured JSON outputs with confidences. Every agent decision is deterministically reproducible.

Gasless execution via sponsored transactions. Multi-step strategies atomically in one Programmable Transaction Block. Agent-to-agent payments via Google AP2.
