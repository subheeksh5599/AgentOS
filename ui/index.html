04 — pipeline

How an agent thinks.

01

Fetch State SUI RPC

Agent queries Sui testnet for live pool APRs, order book spreads, and prediction market odds. Real on-chain data only.

02

LLM Reason GROQ — LLAMA 3.3 70B

State is sent to Groq with the agent's strategy prompt. The LLM returns a structured JSON decision — which pool, how much, at what confidence.

03

Guardrail Check ON-CHAIN

Decision is validated against programmable guardrails: max TX value, daily spend limit, allowed action types. Blocked actions are logged, never executed.

04

Execute SPONSORED TX

Valid decisions are packaged into a Programmable Transaction Block and submitted to Sui testnet. Gas is sponsored — the agent pays nothing.

05

Log to Walrus IMMUTABLE

Every decision — action, reasoning, hash, timestamp — is uploaded to Walrus as a content-addressed blob. Verifiable, tamper-proof, permanent.

05 — architecture

Move contracts.
Seven modules.

01

agent_registry.move DISCOVERY

Central directory for all deployed agents. Owner binding, category indexing, active/paused toggle. Emits AgentRegistered events for off-chain indexing.

02

agent_wallet.move GUARDRAILS

Per-agent wallet with configurable constraints: max TX value, daily spend limit, whitelisted action types. Only the capability holder can authorize agent actions.

03

agent_factory.move 1-CLICK DEPLOY

Creates wallet + registers agent + funds + sets guardrails in one Programmable Transaction Block. Emits AgentDeployed event.

04

action_log.move AUDIT TRAIL

Append-only log with Walrus blob IDs and Sui TXN digests. Each entry is content-addressed. Can be sealed to a Walrus batch for permanent storage.

05

yield_agent.move EXAMPLE

Yield optimization strategy: pool scanning, APR comparison, rebalancing thresholds, compound logic. Configurable min APR and max single-pool exposure.

06

trader_agent.move EXAMPLE

Arbitrage strategy: order book analysis, stop-loss/take-profit triggers, pair whitelisting, position tracking with signed PnL.

07

prediction_agent.move EXAMPLE

Prediction market strategy: Kelly criterion sizing, confidence thresholds, edge detection, auto-compound winnings to next market.
