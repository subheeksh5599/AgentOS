Sui Primitives

7 Sui primitives powering every agent.

🌊

Walrus

Verifiable agent memory, execution logs, model configs stored as content-addressed blobs.

🔐

Seal

Encrypted agent state and private keys — agents hold funds without exposing keys on-chain.

📚

DeepBook

Shared liquidity across Spot, Margin, and Predict markets. Agents trade via unified interface.

🆔

zkLogin

Agent identity tied to real users via Google OAuth — not just a keypair floating in space.

⛽

Sponsored TXs

Gasless agent actions — agents execute autonomously without needing SUI in their wallet.

⊞

PTBs

Atomic multi-step strategies in one transaction — swap→stake→log in a single block.

💳

Google AP2

Agent-to-agent payments on Sui (Google's launch partner). Agents pay each other.
